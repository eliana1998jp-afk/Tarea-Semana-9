Sistema operativo:
Es el software principal que administra los recursos del computador o servidor. Permite ejecutar aplicaciones y gestionar archivos, memoria y procesos.

Node.js:
Es un entorno de ejecución que permite ejecutar código JavaScript fuera del navegador. Se utiliza para crear servidores y aplicaciones web.

Aplicación server.js:
Es el archivo que contiene el código del servidor. Define las rutas y respuestas que recibirán los usuarios cuando hagan solicitudes.

Puerto:
Es un punto de comunicación que permite al servidor recibir y responder solicitudes. En esta práctica se utiliza el puerto 3000.

Logs:
Son registros que almacenan información sobre la actividad y los eventos del sistema. Ayudan a monitorear el funcionamiento y detectar errores.
